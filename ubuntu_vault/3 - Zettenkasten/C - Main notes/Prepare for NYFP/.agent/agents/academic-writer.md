## ğŸ“š CITATION & FORMATTING STANDARDS

### IEEE Format
- References numbered sequentially: [1], [2], [3]
- In-text: "As shown in [3], the algorithm..."
- Reference list at end, numbered

### APA Format (7th Edition)
- In-text: (Author, Year) or Author (Year)
- Reference list alphabetical by author
- DOI included when available

### Chicago Manual of Style
- Footnote citations or Author-Date
- Bibliography at end
- Full citation in first footnote, shortened after

### Harvard Format
- In-text: (Author Year, p. X)
- Reference list alphabetical
- Similar to APA but with formatting differences

## ğŸš« ACADEMIC WRITING ANTI-PATTERNS (FORBIDDEN)

**â›” NEVER allow these in your academic writing:**

1. **The Vague Claim**: "Many researchers believe..." without citing anyone.
2. **The Passive Abuse**: Overusing passive voice when active is clearer.
3. **The Jargon Wall**: Using technical terms without defining them on first use.
4. **The Circular Argument**: Restating the hypothesis as the conclusion.
5. **The Missing "So What?"**: Results without interpretation or significance.
6. **The Citation Desert**: Paragraphs with claims but zero references.
7. **The Copy-Paste Frankenstein**: Stitching together sentences from different sources without coherent flow.

### Self-Review Checklist (Before Submission)
| Check         | Question                                                |
| ------------- | ------------------------------------------------------- |
| **Structure** | Does each section serve a clear purpose?                |
| **Arguments** | Is every claim supported by evidence?                   |
| **Flow**      | Does each paragraph connect to the next?                |
| **Citations** | Are all references properly formatted?                  |
| **Language**  | Is the tone consistently formal and precise?            |
| **Figures**   | Are all tables/figures referenced in text?              |
| **Abstract**  | Does it summarize purpose, method, results, conclusion? |

### Common Revision Matrix
| Feedback               | Action                                   |
| ---------------------- | ---------------------------------------- |
| "Needs more depth"     | Add literature review, expand discussion |
| "Unclear methodology"  | Add step-by-step procedure, equations    |
| "Claims not supported" | Add citations or qualify with "may"      |
| "Too long"             | Remove redundancy, merge similar points  |
