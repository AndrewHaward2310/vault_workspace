# CV Section Patterns

## Writing Impact-Driven CV Sections

### The STAR-Q Method for Experience Bullets
**S**ituation → **T**ask → **A**ction → **R**esult → **Q**uantify

```
❌ Weak: "Responsible for managing the team."
✅ Strong: "Led a cross-functional team of 8 engineers to deliver 
   a real-time data pipeline, reducing processing latency by 40%."
```

### Action Verbs by Category

| Category          | Verbs                                                      |
| ----------------- | ---------------------------------------------------------- |
| **Leadership**    | Led, Directed, Managed, Coordinated, Supervised            |
| **Technical**     | Developed, Engineered, Implemented, Architected, Optimized |
| **Research**      | Investigated, Analyzed, Evaluated, Discovered, Published   |
| **Communication** | Presented, Authored, Collaborated, Mentored, Trained       |
| **Achievement**   | Achieved, Increased, Reduced, Improved, Exceeded           |

### Section-Specific Patterns

#### Education
```
[Degree] in [Field], [University], [City]              [Year]
• GPA: X.XX/4.00 (include if > 3.5)
• Thesis: "Title" (if applicable)
• Relevant Coursework: [only if limited experience]
• Dean's List / Honors (if applicable)
```

#### Publications (Academic CV)
```
Journal Articles:
[N] Author, A., Your Name, & Co-Author. (Year). "Title." 
    Journal, Vol(Issue), pp. doi: X

Conference Papers:  [list separately]
Preprints:          [list separately]
Patents:            [list separately]
```

#### Skills Section
```
Programming Languages: Python (expert), C++ (proficient), SQL
ML/AI: PyTorch, TensorFlow, scikit-learn, LangChain
DevOps: Docker, Kubernetes, AWS (EC2, S3, Lambda)
Languages: Vietnamese (native), English (fluent, IELTS 7.5)
```
