# Bilingual CV Guide (English - Vietnamese)

## Creating Professional Bilingual Career Documents

### When to Use Bilingual Format
- Vietnamese companies with international partners
- Scholarship applications requiring both languages
- Multinational organizations in Vietnam
- Academic positions at Vietnamese universities

### Formatting Approaches

**Option A: Side-by-Side** (for 1-page resumes)
```
ENGLISH SECTION          |  PHẦN TIẾNG VIỆT
─────────────────────────|──────────────────────
Education                |  Học vấn
Experience               |  Kinh nghiệm
Skills                   |  Kỹ năng
```

**Option B: Sequential** (recommended for multi-page CVs)
```
[English Version — Pages 1-3]
[Vietnamese Version — Pages 4-6]
```

### Key Term Translations

| English              | Vietnamese               |
| -------------------- | ------------------------ |
| Education            | Học vấn                  |
| Work Experience      | Kinh nghiệm làm việc     |
| Research Experience  | Kinh nghiệm nghiên cứu   |
| Publications         | Công bố khoa học         |
| Awards & Honors      | Giải thưởng và Danh hiệu |
| Skills               | Kỹ năng                  |
| Professional Summary | Tóm tắt chuyên môn       |
| References           | Người tham khảo          |
| Teaching Experience  | Kinh nghiệm giảng dạy    |
| Certifications       | Chứng chỉ                |
| Projects             | Dự án                    |

| Bachelor's degree    | Cử nhân                  |
| Master's degree      | Thạc sĩ                  |
| Ph.D. / Doctorate    | Tiến sĩ                  |
| GPA                  | Điểm trung bình          |

### Tips
1. Keep technical terms in English (PyTorch, React, etc.)
2. University names: use official bilingual name if available
3. Job titles: provide both but prioritize the language of the target audience
4. Dates: use international format (MM/YYYY) for consistency
