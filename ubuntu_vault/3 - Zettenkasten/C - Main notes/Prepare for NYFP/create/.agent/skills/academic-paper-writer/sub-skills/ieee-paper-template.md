# IEEE Paper Template

## IEEE Conference / Journal Paper Format

### Page Layout
- **Page size**: US Letter (8.5" × 11") or A4
- **Columns**: Two-column format
- **Margins**: Top 0.75", Bottom 1", Left/Right 0.625"
- **Font**: Times New Roman, 10pt body text
- **Title**: 24pt, centered, bold

### Template Structure

```
TITLE OF PAPER (24pt, Bold, Centered)

Author Name¹, Co-Author Name²
¹Department, University, City, Country
²Department, University, City, Country
author@email.com

Abstract— [150-200 words, single paragraph, italic.
State the problem, approach, key results, and significance.]

Index Terms— keyword1, keyword2, keyword3, keyword4

I. INTRODUCTION
[Section headers: Roman numerals, ALL CAPS, centered]

   A. Subsection Title
   [Subsection headers: Capital letter, Italic, left-aligned]

II. RELATED WORK

III. PROPOSED METHOD
   A. System Architecture
   B. Algorithm Description

IV. EXPERIMENTAL SETUP
   A. Dataset
   B. Evaluation Metrics
   C. Baselines

V. RESULTS AND DISCUSSION
   A. Main Results
   B. Ablation Study

VI. CONCLUSION

ACKNOWLEDGMENT
[Optional, no section number]

REFERENCES
[1] A. Author, "Title of paper," Journal Name, vol. X, 
    no. Y, pp. Z1-Z2, Month Year.
[2] B. Author and C. Author, "Title," in Proc. Conf. Name, 
    City, Country, Year, pp. Z1-Z2.
```

### IEEE Citation Format
```
In-text:  [1], [2], [1]-[3], [1, 4, 7]

Journal:  [1] A. B. Author, "Title of article," Title of Journal,
          vol. X, no. Y, pp. Z1–Z2, Abbrev. Month, Year.

Conference: [2] A. Author, "Title of paper," in Proc. Name of Conf.,
            City, State/Country, Year, pp. X–Y.

Book:     [3] A. Author, Title of Book, xth ed. City, State/Country: 
          Publisher, Year.

Website:  [4] A. Author. "Title of page." Website Name. URL 
          (accessed Mon. Day, Year).
```

### Key IEEE Rules
1. Figures and tables numbered separately (Fig. 1, TABLE I)
2. Table captions ABOVE the table
3. Figure captions BELOW the figure
4. References numbered in order of first appearance
5. No footnotes (use references or inline text instead)
