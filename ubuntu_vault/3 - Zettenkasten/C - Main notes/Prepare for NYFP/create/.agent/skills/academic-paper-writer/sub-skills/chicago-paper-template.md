# Chicago Paper Template (17th Edition)

## Chicago Manual of Style — Notes-Bibliography System

### Page Layout
- **Font**: Times New Roman 12pt
- **Spacing**: Double-spaced text, single-spaced block quotes
- **Margins**: 1 inch all sides
- **Page numbers**: Bottom center or top right

### Template Structure

```
                        Title of Paper
                     Subtitle If Needed

                       Author Name
                    Course / Department
                     University Name
                     Date of Submission


                       Introduction

    The introduction provides context and states the thesis.¹

                    Section Heading

    Body paragraphs with footnote citations.²

                    Another Section

    Continue with analysis and evidence.³

                      Conclusion

    Summarize findings and significance.

                        Notes

1. Author First Last, Title of Book (Place: Publisher, Year), page.
2. Author First Last, "Title of Article," Journal Name Vol, 
   no. Issue (Year): pages.
3. Shortened reference: Last, Short Title, page.

                     Bibliography

Author Last, First. Title of Book. Place: Publisher, Year.

Author Last, First. "Title of Article." Journal Name Vol, 
    no. Issue (Year): page range.
```

### Chicago Citation Format (Notes-Bibliography)

```
FOOTNOTES:
First reference (full):
  1. Author First Last, Title of Book (Place: Publisher, Year), page.
  
Subsequent references (shortened):
  2. Last, Short Title, page.

Journal article:
  3. Author First Last, "Article Title," Journal Title Vol, no. Issue
     (Year): pages.

BIBLIOGRAPHY:
Book:       Last, First. Title of Book. Place: Publisher, Year.
Article:    Last, First. "Article Title." Journal Title Vol, no. Issue
            (Year): page–page.
Chapter:    Last, First. "Chapter Title." In Book Title, edited by 
            Editor First Last, pages. Place: Publisher, Year.
Website:    Last, First. "Page Title." Website Name. Last modified 
            Date. URL.
```

### Key Chicago Rules
1. Footnotes numbered consecutively throughout the paper
2. Bibliography entries alphabetical by author last name
3. Bibliography has hanging indent
4. Book titles in *italics*; article titles in "quotes"
5. Use "Ibid." for consecutive references to same source
6. Block quotes (100+ words) indented, no quotation marks
