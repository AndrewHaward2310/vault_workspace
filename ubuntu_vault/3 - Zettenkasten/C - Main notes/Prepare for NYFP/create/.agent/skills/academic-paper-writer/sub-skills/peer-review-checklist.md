# Peer Review Checklist

## Self-Review Before Submission

### Structure & Organization
- [ ] Title is specific and descriptive
- [ ] Abstract follows the 4-element pattern (context, method, results, significance)
- [ ] Introduction clearly states the research gap
- [ ] Related work covers all major relevant studies
- [ ] Methodology is detailed enough for reproduction
- [ ] Results are presented before interpretation
- [ ] Discussion connects findings to existing literature
- [ ] Conclusion summarizes contributions and future work

### Technical Quality
- [ ] All claims are supported by evidence or citations
- [ ] Statistical tests are appropriate and correctly reported
- [ ] Figures are clear, labeled, and referenced in text
- [ ] Tables have consistent formatting
- [ ] All equations are numbered and explained
- [ ] Ablation studies validate component contributions

### Writing Quality
- [ ] Consistent verb tense (past for methods/results, present for facts)
- [ ] No grammar or spelling errors
- [ ] Paragraphs flow logically with clear transitions
- [ ] No undefined abbreviations or jargon
- [ ] Active voice preferred over passive
- [ ] Within page/word limit

### Citations & References
- [ ] All references are cited in text
- [ ] All in-text citations appear in reference list
- [ ] Citation format is consistent (IEEE/APA/Chicago/Harvard)
- [ ] Recent references included (within last 3-5 years)
- [ ] DOIs included where available
- [ ] Self-citations are proportionate

### Formatting
- [ ] Follows target venue template exactly
- [ ] Page margins, font, and spacing are correct
- [ ] Figure/table numbering is sequential
- [ ] Appendix referenced from main text
- [ ] Author information matches submission system
