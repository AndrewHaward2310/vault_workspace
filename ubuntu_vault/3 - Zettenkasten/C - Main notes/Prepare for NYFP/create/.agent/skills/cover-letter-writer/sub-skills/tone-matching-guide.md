# Tone Matching Guide

## Adapting Writing Tone to Your Audience

### Tone Spectrum

| Tone Level                      | When to Use                                | Characteristics                                    |
| ------------------------------- | ------------------------------------------ | -------------------------------------------------- |
| **Highly Formal**               | Academic journals, government applications | No contractions, passive voice OK, technical terms |
| **Professional**                | Job applications, business cover letters   | Active voice, clear, concise, no slang             |
| **Conversational Professional** | Startup applications, personal blogs       | Friendly but competent, light humor OK             |
| **Personal**                    | Personal statements, essays                | Authentic voice, stories, emotions OK              |

### Organization-Specific Tone Matching

| Organization Type          | Tone                | Approach                                   |
| -------------------------- | ------------------- | ------------------------------------------ |
| **Traditional University** | Formal academic     | Emphasize rigor, methodology, publications |
| **Tech Company**           | Professional-casual | Show impact, use metrics, be direct        |
| **Startup**                | Energetic, direct   | Show initiative, passion, speed            |
| **NGO/Foundation**         | Mission-driven      | Connect to values, social impact           |
| **Government**             | Formal, structured  | Follow format exactly, be thorough         |

### Quick Tone Checks
- Read the organization's website — mirror their language style
- Look at their job posting tone — formal or casual?
- Check their social media — how do they communicate?
- If in doubt, err on the side of slightly more formal
