# Readability Scoring

## Measuring Writing Clarity

### Common Readability Metrics

| Metric                   | Formula Basis                   | Target for Academic Writing |
| ------------------------ | ------------------------------- | --------------------------- |
| **Flesch Reading Ease**  | Sentence length + syllables     | 30-50 (Difficult)           |
| **Flesch-Kincaid Grade** | Same factors → grade level      | 12-16 (College+)            |
| **Gunning Fog Index**    | Complex words + sentence length | 12-17                       |
| **SMOG Index**           | Polysyllabic words              | 12-16                       |
| **Coleman-Liau Index**   | Characters per word             | 12-16                       |

### Interpreting Flesch Reading Ease

| Score  | Level          | Appropriate For      |
| ------ | -------------- | -------------------- |
| 90-100 | Very Easy      | Children's reading   |
| 60-70  | Standard       | General audience     |
| 30-50  | Difficult      | Academic papers ✅    |
| 0-30   | Very Difficult | Legal/technical docs |

### Improving Readability
1. **Shorten sentences**: Target 15-25 words per sentence
2. **One idea per paragraph**: Don't overload
3. **Use topic sentences**: First sentence = paragraph main point
4. **Active voice when possible**: Clearer and more direct
5. **Avoid nested clauses**: Restructure complex sentences

### Readability Checklist
- [ ] Average sentence length < 25 words
- [ ] Paragraphs are 3-8 sentences
- [ ] Key terms defined on first use
- [ ] Transitions between paragraphs are smooth
- [ ] No wall-of-text paragraphs (> 10 sentences)
