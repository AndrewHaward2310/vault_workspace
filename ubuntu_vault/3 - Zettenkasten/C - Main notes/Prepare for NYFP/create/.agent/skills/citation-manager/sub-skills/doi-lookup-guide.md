# DOI Lookup Guide

## Finding and Using DOIs

### What is a DOI?
A **Digital Object Identifier** is a permanent link to a scholarly document.
Format: `10.XXXX/XXXXX` → URL: `https://doi.org/10.XXXX/XXXXX`

### How to Find DOIs

1. **CrossRef**: https://search.crossref.org — Search by title, author, or metadata
2. **DOI.org**: https://doi.org — Resolve existing DOIs to their source
3. **Google Scholar**: Click "Cite" → look for DOI in citation
4. **Paper itself**: Usually on the first page, near the journal info

### DOI to Citation Conversion
Use DOI to auto-generate formatted citations:
- **doi2bib**: https://www.doi2bib.org — DOI → BibTeX
- **Citation Machine**: Auto-format in APA, MLA, Chicago
- **Zotero/Mendeley**: Paste DOI to auto-import

### Including DOIs in References

| Format      | Example                                    |
| ----------- | ------------------------------------------ |
| **IEEE**    | Available: https://doi.org/10.1234/example |
| **APA**     | https://doi.org/10.1234/example            |
| **Chicago** | https://doi.org/10.1234/example            |
| **Harvard** | doi:10.1234/example                        |
| **BibTeX**  | `doi = {10.1234/example}`                  |

### Tips
- Always prefer DOI over URL — DOIs are permanent
- If no DOI exists, check if the publisher assigned one recently
- Use the `https://doi.org/` prefix (not `dx.doi.org`)
