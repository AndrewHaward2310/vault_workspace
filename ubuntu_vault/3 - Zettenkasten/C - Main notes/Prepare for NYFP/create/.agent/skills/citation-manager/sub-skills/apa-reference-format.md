# APA Reference Format (7th Edition)

## APA Citation Quick Reference

### In-Text Citations
- **Parenthetical**: (Author, Year) — "Results were significant (Smith, 2024)."
- **Narrative**: Author (Year) — "Smith (2024) found that..."
- **Two authors**: (Smith & Jones, 2024) or Smith and Jones (2024)
- **Three+**: (Smith et al., 2024) from first citation
- **Direct quote**: (Smith, 2024, p. 15)
- **Multiple**: (Smith, 2024; Jones, 2023) — alphabetical order

### Reference List Entries

```
Journal Article:
Author, A. A., & Author, B. B. (Year). Title of article. Title of 
    Periodical, Volume(Issue), Page–Page. https://doi.org/xxxxx

Book:
Author, A. A. (Year). Title of work: Capital letter for subtitle 
    (Xth ed.). Publisher. https://doi.org/xxxxx

Edited Book Chapter:
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), 
    Title of book (Xth ed., pp. XX–XX). Publisher. 
    https://doi.org/xxxxx

Conference Paper:
Author, A. A. (Year, Month Days). Title of paper [Conference 
    presentation]. Name of Conference, City, State/Country.

Thesis/Dissertation:
Author, A. A. (Year). Title of thesis [Doctoral dissertation, 
    University Name]. Database Name. https://doi.org/xxxxx

Website:
Author, A. A. (Year, Month Day). Title of page. Site Name. 
    https://www.url.com

Software:
Author, A. A. (Year). Title of software (Version X.X) [Computer 
    software]. https://www.url.com
```

### APA 7th Edition Key Changes
1. Use "et al." from first citation for 3+ authors
2. DOIs formatted as hyperlinks: https://doi.org/xxxxx
3. No "Retrieved from" for most online sources
4. Include up to 20 authors before using "..." 
5. Include issue number for ALL journal articles
