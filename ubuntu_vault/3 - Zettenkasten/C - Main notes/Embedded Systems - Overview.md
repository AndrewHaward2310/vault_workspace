# Embedded Systems - Overview

# 1. What is embedded system? 
- refers to something that is joined to another entity
- An embedded system can function as a standalone unit or as a component of a larger system
- a microcontroller or microprocessor based system that is intended to carry out a certain function is called an embedded system
- An embedded system consists of 3 parts:
	- Hardware is present
	- Application are available for it 
	- It has a Real Time Operating System (RTOS) that manages the application software and offers a way to schedule and allow the processor to execute a process by adhering to a latency control plan
