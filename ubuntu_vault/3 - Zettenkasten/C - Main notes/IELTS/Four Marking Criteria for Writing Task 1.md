# Four Marking Criteria for Writing Task 1

## Task Achievement
- Analyse the chart and identify all key features
- Highlight your key features in an overview
- Support your body language paragraph with data (numbers and/or dates)
- Avoid opinion / write a factual report 
- Avoid putting in information which is wrong 
- Write over 150 words

## Coherence and Cohesion
based on the organization of information,  the paragraph structure and the use of linking devices to mark 
- plan where to put your information
- have four body paragraphs
- use a range of linking devices
- avoid mistakes with linking works
- use referencing (this, it, ... )
![[Pasted image 20251110230414.png]]
## Lexical Resource 
vocabulary, spelling, and the number of mistakes are makred
- use a range of vocabulary, especially to present accurate data
- avoid spelling mistakes
- avoid using the wrong words or informal language

## Grammar
- a range of sentence structures 
- right tense
- right word order
- avoid mistakes
- right punctuation
